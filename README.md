# ⏰ Alarm Clock Web App

A simple and user-friendly Alarm Clock Web Application built using **HTML, CSS, and JavaScript**. This app allows users to set alarms and get notified at the selected time.

---

## 🚀 Features

- ⏰ Set alarm for a specific time
- 🔔 Alarm sound notification
- 🕒 Real-time clock display
- 🎨 Clean and responsive UI
- ❌ Option to clear alarm
